# Raw Sources

This directory stores immutable input material for the second brain.

- `inbox/` is for newly dropped sources awaiting ingest.
- `processed/` is for sources that have already been incorporated.
- `assets/` is for local images, PDFs, and downloaded attachments referenced by sources.
- Podcast transcript markdown files may also live directly under `raw/` when generated from RSS feeds.

The LLM may read files here, but should not rewrite them.

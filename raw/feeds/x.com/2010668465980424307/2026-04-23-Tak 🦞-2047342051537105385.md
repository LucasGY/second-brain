# OpenClaw 2026.4.22: A few other things in this release that didn’t make the main tweet:🦞 🖼️ gpt-image-2 is now the default OpenAI image path, ...

**@Tak 🦞** · 2026-04-23 15:49

OpenClaw 2026.4.22: A few other things in this release that didn’t make the main tweet:🦞

🖼️ gpt-image-2 is now the default OpenAI image path, so your lobster’s image output got a nice little glow-up without you having to touch config first.
🔊 xAI picked up image generation, TTS, STT, and live transcription support, so Grok can finally contribute to the group project.
🎙️ Realtime transcription expanded to Deepgram, ElevenLabs, and Mistral, because having more than one serious voice lane is just objectively nicer.
🛠️ /models add   lets you register a model from chat without restarting, because rebooting your gateway for every tiny experiment is a bit ridiculous.
🧠 GPT-5 behavior tuning now carries across more GPT providers, so the better personality + better follow-through upgrade is no longer stuck in one special-case lane.

# PE 和Forward PE！盈利泡沫论

**@投资TALK君** · 2026-05-09 17:15

股市交易的是未来，后视镜里的TTM PE才是最没有意义的。
最近有一个被讨论很多的观点，现在的泡沫不在Forward PE，而在Forward EPS，股价 = 估值*盈利，泡沫在未来盈利，不是估值。这是一个正确的视角，那么看一下最近涨得好的公司，未来一年的盈利是不是有风险，订单是不是确定？订单来自于谁？他们的客户现金流有没有风险？这才是问题的核心。
1. 芯片，内存，光，储能，未来一年订单基本都是满的。所以Forward EPS确定性很高
2. 他们的客户有没有可能跳票？客户是谁？大科技。跳票可能性高吗？我们认为他们资本开支会减少吗？我看不到减少的可能。谷歌罕见地在一季度财报后就给了27年的资本开支指引。较大幅度高于26年
3. 大科技的业务，现金流有没有问题？这个问题的核心是他们原来的资本开支有没有回报，答案反应在今年的营收有没有加速，关键数据运营盈利有没有加速，财报看到是有的。另外就是他们的订单有没有增加，答案是暴增
4. 大科技的多数订单来自于谁？大模型。大模型有钱吗？有。有业务落地吗？很快。营收增速几个月就翻4倍，还是大基数的前提下。大模型的营收来自于哪？API调用，企业订阅。
所以核心到了哪？AI的落地。至今为止多数落地还是在coding，而coding的市场很大，所以营收增速快。其次coding又是其他产品开发的工具，所以期待AI在其他行业和领域的落地，和智能体的落地。
我们的确可以很轻易地下结论说未来的盈利是泡沫，却忽略了和00年最大的差别：客户质量+产品落地。00年的科技泡沫是估值+盈利的泡沫，现在的市场多数估值绝对不存在泡沫，100，200P/E的并不常见，FORWARD EPS的泡沫必须紧跟AI的每个落地，找到核心的判断方向。

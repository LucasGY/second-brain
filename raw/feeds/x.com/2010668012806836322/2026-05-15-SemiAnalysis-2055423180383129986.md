# Re DeepSeek takes it one step further by breaking up the workload and adding fine-grained pipelining of communication and computation within a single ...

**@SemiAnalysis** · 2026-05-15 23:00

Re DeepSeek takes it one step further by breaking up the workload and adding fine-grained pipelining of communication and computation within a single layer. MegaMoE breaks the experts into waves, and instead of only overlapping TMA memory loads+tensor core math+epilogue SIMT work, the kernel pipelines and overlaps NVLink communication, memory loads, and computation for each wave of experts. The scheduler is a state machine that alternates Linear-1->Linear-2 within each wave, then advances to the next.

Synchronization between stages uses lightweight barriers rather than kernel launches. Dispatch writes a token and increments an arrival counter. TMA load waits on that counter, then begins loading the dispatched tokens. After the L1 SwiGLU epilogue, the block arrives at Linear 2 barrier and the expert begins to compute Linear 2. (3/4)

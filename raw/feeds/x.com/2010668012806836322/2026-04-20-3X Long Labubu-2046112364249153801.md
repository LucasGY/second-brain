# RT Aaron: The entire model hinges on a single assumption: how much server CPU ASPs can rise. On Intel's side, the Xeon revenue model is +36% YoY, with...

**@3X Long Labubu** · 2026-04-20 05:28

RT Aaron
The entire model hinges on a single assumption: how much server CPU ASPs can rise. On Intel's side, the Xeon revenue model is +36% YoY, with gross margin exit rate pulled from 40% to 45% by year-end, almost entirely driven by ASP. The problem is, there's a chart in the report breaking down $INTC server CPU shipment structure—Sapphire Rapids and Emerald Rapids, those 10nm/7nm legacy node products, dominate the mix, while Granite Rapids has a very small share. With customers snapping up whatever compute they can get amid shortages, the quality of this pricing power is worlds apart from the pricing power AMD gains from steadily eating market share at the architecture level with EPYC.

On $AMD's side, the 2027 story relies almost entirely on $META. After folding Meta into the model, 2027 AI revenue jumps from $27B to $43B, putting the total revenue model $9B+ above consensus. But he also leaves an honest question hanging. AMD hasn't yet proven it can land large AI orders without handing out company warrants. If every deal needs a warrant to close, the quality of that growth gets marked down.

These analyses are all solid, but Rasgon might be underestimating the explosive demand for CPUs from agentic AI. He keeps repeating "agentic server CPU surge" without breaking down where this demand is really coming from. The current CPU supply-demand imbalance is mostly from GPU bundling—every AI chip comes with a fixed ratio of CPUs, essentially hardware-tied demand. On the software side, standalone CPU compute from agentic API calls is close to zero right now. In other words, agent calls haven't even taken off yet, and CPUs are already up 20% in price with supply shortages.

So what happens when agents truly scale? Each GPU generation doubles token throughput, meaning each GPU can support more agents simultaneously, and the corresponding CPU thread demand expands exponentially. Agent coding call volumes are climbing at 10x per year. The CPU demand driver is shifting from "GPU bundling ratios" to "structural pull from software API calls." Rasgon's 2027 modeling still follows a gentle ASP ramp, but if agentic API calls scale along the current trajectory, the question for CPUs won't be "how long can pricing hold"—it'll be becoming a structural bottleneck for the entire AI infrastructure. His model might not be too optimistic; it might not be optimistic enough.

Intel reports earnings on April 23—near-term, watch ASP and PC. But the real thing to track is the acceleration in agentic call volumes; that's the true far-end variable for CPU pricing power.

# Definition
Microsoft is a cloud and software platform company whose Azure, Microsoft 365, Copilot, and enterprise distribution make it central to AI workflow adoption.
> Microsoft 是一家云和软件平台公司，其 Azure、Microsoft 365、Copilot 和企业分发能力使其成为 AI 工作流 adoption 的核心参与者。

## Strategic / Technical Significance
* Microsoft matters because it can package AI agents inside existing enterprise workflows and shift software monetization from seats toward productivity and units of work.
  > Microsoft 重要，因为它可以把 AI Agent 嵌入既有企业工作流，并推动软件变现从席位转向生产力和工作单元。

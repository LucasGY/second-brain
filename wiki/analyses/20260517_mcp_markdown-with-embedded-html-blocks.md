## Summary
A second-brain note can keep Markdown as the durable source while embedding an HTML reading card for scanning, visual hierarchy, and richer presentation.
> Second Brain 笔记可以保留 Markdown 作为长期知识源，同时嵌入 HTML 阅读卡片，用于快速浏览、视觉层级和更丰富的呈现。

## Knowledge
The practical pattern is not to replace Markdown entirely. Markdown remains the canonical layer for YAML frontmatter, Obsidian wikilinks, grep, Git diffs, and agent editing. The HTML block is a presentation layer placed near the top of the note. It can summarize the note, expose tags, show related pages, and make dense content easier to scan. This follows the spirit of the recent HTML artifact discussion: when AI produces long outputs, humans benefit from structure, layout, and visual affordances instead of a flat wall of text.
> 实用模式不是完全替代 Markdown。Markdown 继续作为 YAML frontmatter、Obsidian wikilink、grep、Git diff 和 agent 编辑的规范层。HTML 块则作为展示层，放在笔记顶部附近。它可以概括笔记、展示标签、列出相关页面，并让密集内容更容易浏览。这符合最近 HTML artifact 讨论的精神：当 AI 产出很长时，人类需要结构、布局和视觉提示，而不是一整面扁平文本墙。

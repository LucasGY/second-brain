# Core Definition
Platform Shifts occur when a dominant technology platform is replaced by a new paradigm, fundamentally changing how users interact with technology and how value is created and captured. These shifts favor builders who adopt new tools early, while incumbents often struggle to adapt.
Platform Shifts（平台转移）发生在主导技术平台被新范式取代时，从根本上改变用户与技术交互的方式，以及价值创造和捕获的方式。这些转移有利于早期采用新工具的构建者，而现有企业往往难以适应。

## 🛠️ Mechanisms & Architecture

**Characteristics of Platform Shifts:**

1. **Opportunity Window:** Early movers during shifts capture disproportionate value (e.g., CUDA developers in 2010s AI wave).
2. **Ecosystem Lock-in:** New platforms build ecosystems that become defensible moats.
3. **Incumbent Disruption:** Companies too aligned with prior platform struggle (e.g., Sony missed mobile).
4. **Software Commoditizes Hardware:** As hardware commoditizes, software differentiates.

**Historical Examples:**
- Mainframe → PC (IBM, Apple)
- PC → Internet (Amazon, Google)
- Internet → Mobile (Apple, Google, Android)
- Mobile → AI (NVIDIA, OpenAI)

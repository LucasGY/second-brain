# New Zealand RV → PRV → Citizenship: Annotated Flowchart HTML

This note replaces the earlier text-mirroring HTML approach. The intended HTML artifact should work like an annotated flowchart rather than a long article.

## Display model

The HTML page should have these sections:

1. Hero / TL;DR area - Title: New Zealand RV to PRV to Citizenship: Clickable Decision Flow. - Four prominent numbers: 2y, 184, 240, 1350. - Short statement: PRV does not normally reset the citizenship clock.

2. Main annotated flowchart - Left/main area: clickable flow nodes. - Right side: detail panel that changes when a node is clicked. - Legend: process, decision, pass path, fail path, terminal.

Knowledge Body / 知识正文

The note should not depend on raw inline HTML alone. A standalone HTML artifact is closer to the HTML effectiveness pattern because it can carry layout, responsive CSS, and future JavaScript without fighting Markdown sanitization. The Markdown page remains the canonical editable record, while Custom Frames turns the HTML artifact into an embedded reading surface inside Obsidian.

笔记不应只依赖原始内嵌 HTML。独立 HTML artifact 更接近 HTML effectiveness 的模式，因为它可以承载布局、响应式 CSS 和未来的 JavaScript，而不用和 Markdown 的安全过滤机制对抗。Markdown 页面仍然是可编辑的规范记录，Custom Frames 则把 HTML artifact 变成 Obsidian 内部的嵌入式阅读界面。

# New Zealand RV to PRV to Citizenship: Obsidian Flowchart Note

> [!info] Checked date > Checked on 2026-05-17 against official New Zealand immigration/government guidance. Use this as a planning note, not legal advice.

## One-line memory

> [!tip] > **PRV** = 2 years as resident + 184 days in each of the two immediately preceding 12-month periods. > **Citizenship** = 5 years qualifying resident status + 240 days in each 12-month period + 1,350 days total. > **PRV does not normally reset the citizenship clock. RV/resident time can count.**

## Main decision flow

```mermaid flowchart TD A[Start: Resident Visa / resident status begins] --> B{Has resident status reached 2 years?} B -- No --> B1[Usually not ready for PRV yet, even if the second-year 184 days are already reached] B -- Yes --> C{PRV test: in the 24 months before application, are there 184+ resident days in each 12-month block?} C -- No --> C1[Keep accumulating days or choose a later application date] C -- Yes --> D[Possible PRV application point] D --> E[Continue accumulating qualifying resident time] E --> F{Citizenship test: in the 5 years before application, are there 240+ days in each 12-month block and 1,350+ days total?} F -- No --> F1[Keep accumulating days; long absences may push the date later] F -- Yes --> G[Possible citizenship by grant application point] ```

# 新西兰 RV 到 PRV 再到国籍：Obsidian 流程图笔记

> [!info] 核对日期 > 本笔记按 2026-05-17 核对的新西兰移民局/新西兰政府官方规则整理。用于规划，不替代法律或移民建议。

## 一句话速记

> [!tip] > **PRV** = resident 身份满 2 年 + 申请日前倒推两个 12 个月区间各 184 天。 > **国籍** = 过去 5 年合格 resident 身份 + 每个 12 个月区间 240 天 + 5 年合计 1,350 天。 > **PRV 通常不会重置国籍 5 年计时；RV/resident 时间通常可以算。**

## 总流程图

```mermaid flowchart TD A[起点：拿到 Resident Visa / 开始 resident 身份] --> B{resident 身份是否已满 2 年？} B -- 否 --> B1[通常还不能申请 PRV，即使第二年已经住满 184 天] B -- 是 --> C{PRV 检查：申请日前过去 24 个月，两个 12 个月区间是否各有 184+ 天？} C -- 否 --> C1[继续累计天数，或选择更晚的申请日重新倒推] C -- 是 --> D[可能达到 PRV 申请时间点] D --> E[继续累计合格 resident 时间] E --> F{国籍检查：申请日前过去 5 年，是否每个 12 个月区间 240+ 天，且总计 1,350+ 天？} F -- 否 --> F1[继续累计天数；长期离境可能推迟申请时间] F -- 是 --> G[可能达到 citizenship by grant 申请时间点] ```

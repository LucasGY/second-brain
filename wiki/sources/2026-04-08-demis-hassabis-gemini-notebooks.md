# Demis Hassabis: Gemini Notebooks

## Summary | 摘要

Google introduces notebooks in Gemini for project organization. Users can keep multiple projects organized, add past chats and files as sources, and have dedicated workspace.

谷歌在Gemini中推出笔记本功能用于项目组织。用户可以组织多个项目，添加过去的聊天和文件作为来源，拥有专用工作空间。

## Key Features | 关键功能

### Project Organization | 项目组织
- **Multiple notebooks**: Organize by project | 按项目组织
- **Chat history**: Add past conversations | 添加过去对话
- **File sources**: Reference relevant files | 参考相关文件

### Workspace | 工作空间
- **Dedicated space**: Focus on task at hand | 专注手头任务
- **Side panel access**: Easy navigation | 侧边栏访问
- **Context preservation**: Maintain project context | 保持项目上下文

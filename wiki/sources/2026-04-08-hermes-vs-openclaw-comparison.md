# Hermes Agent vs OpenClaw: Full-Dimension Comparison

## Summary | 摘要

A comprehensive comparison between Hermes Agent and OpenClaw across 13 dimensions. Hermes Agent focuses on "agent-first" architecture with self-evolving skills, while OpenClaw emphasizes "control-plane-first" with broader ecosystem capabilities.

Hermes Agent与OpenClaw在13个维度上的全面对比。Hermes Agent聚焦"Agent优先"架构和自进化技能，OpenClaw强调"控制平面优先"和更广泛的生态系统能力。

## Key Findings | 主要发现

### Architecture Philosophy | 架构哲学
- **Hermes**: Single Python Agent as core, "self-evolving work-type Agent"
- **OpenClaw**: Local Gateway/Control Plane as center, "local AI operating system"

### Core Differences | 核心差异

| Dimension | Hermes Agent | OpenClaw |
|-----------|--------------|----------|
| Language | Python | TypeScript |
| Focus | Agent loop, procedural memory | Gateway, multi-agent routing, plugins |
| Complexity | Medium, easy to read | High, multi-layer subsystems |
| Skills | Self-evolving with formal runtime interface | Skills ecosystem with ClawHub distribution |
| Memory | Curated memory, prompt-friendly | Retrieval-based, multi-agent capable |

### Hermes Unique Advantages | Hermes独特优势
1. **Skill Self-Evolution**: Agent has formal write interface to create/edit/patch/delete skills
2. **Transparency**: All work processes visible in chat interface
3. **Long-Running Tasks**: Better at unsupervised autonomous execution
4. **Bayesian Inference**: Continuously learns from mistakes and optimizes workflows

### OpenClaw Unique Advantages | OpenClaw独特优势
1. **Multi-Agent Native**: Multi-brain modeling built into system
2. **Gateway/Platform**: Broader external product surface
3. **Skills Ecosystem**: ClawHub, installer, dangerous-code scanner
4. **Authentication**: Auth profile rotation, failover, model switching

## Use Case Recommendations | 使用场景建议

| Scenario | Better Choice |
|----------|---------------|
| Agent that evolves skills from experience | Hermes |
| Long-term work partner to cultivate | Hermes |
| Multiple agents/channels/accounts | OpenClaw |
| Local control plane integration | OpenClaw |
| Quick hack on core agent | Hermes |
| Skills ecosystem and distribution | OpenClaw |

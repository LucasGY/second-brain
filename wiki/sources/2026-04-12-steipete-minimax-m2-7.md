# Steipete: MiniMax M2.7 Open Source with OpenClaw

## Summary | 摘要

MiniMax M2.7 is now officially open source with SOTA performance on SWE-Pro (56.22%) and Terminal Bench 2 (57.0%). Available via Ollama cloud and OpenClaw integration.

MiniMax M2.7正式开源，在SWE-Pro（56.22%）和Terminal Bench 2（57.0%）上达到SOTA性能。通过Ollama云和OpenClaw集成可用。

## Key Features | 关键功能

### MiniMax M2.7 | MiniMax M2.7
- **SWE-Pro**: 56.22% | 56.22%
- **Terminal Bench 2**: 57.0% | 57.0%
- **License**: Commercial usage | 商业使用许可

### OpenClaw Integration | OpenClaw集成
- **Command**: `ollama launch openclaw --model minimax-m2.7:cloud`
- **Claude integration**: `ollama launch claude --model minimax-m2.7:cloud`

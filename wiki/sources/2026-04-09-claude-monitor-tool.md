# Claude Monitor Tool: Background Agent Scripts

## Summary | 摘要

Claude Monitor Tool lets Claude create background scripts that wake the agent up when needed. Big token saver and moves away from polling.

Claude Monitor Tool让Claude创建在需要时唤醒智能体的后台脚本。大幅节省token，摆脱轮询模式。

## Key Features | 主要功能
- Background script creation
- Wake agent on events
- Log monitoring for errors
- PR polling via script

## Benefits | 优势
- Token savings
- Event-driven vs polling
- More efficient agent loops

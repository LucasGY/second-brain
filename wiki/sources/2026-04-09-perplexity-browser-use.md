# Perplexity Built on Browser Use

## Summary | 摘要

Perplexity discovered to be built on Browser Use open-source library. Hardcoded test prompt "capital of France" found in code. Example of AI startup technical debt.

发现Perplexity基于Browser Use开源库构建。代码中发现硬编码测试提示"法国的首都"。AI创业公司技术债务的案例。

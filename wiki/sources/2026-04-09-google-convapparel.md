# ConvApparel: Human-AI Conversation Dataset

## Summary | 摘要

Google Research releases ConvApparel, a human-AI conversation dataset and evaluation framework to quantify the "realism gap" in LLM-based user simulators.

Google Research发布ConvApparel，一个人机对话数据集和评估框架，用于量化LLM用户模拟器的"现实差距"。

## Purpose | 目的
- Evaluate LLM-based user simulators
- Quantify "realism gap"
- Improve conversational agent training

# Steipete: Anthropic System Prompt Blockers Getting Weirder

## Summary | 摘要

Peter Steinberger comments that Anthropic's system prompt blockers are getting weirder. Third-party apps now draw from extra usage, not from user's plan limits, with Anthropic adding $100 credit for developers.

Peter Steinberger评论Anthropic的系统提示阻止器越来越奇怪。第三方应用现在从额外使用量中扣除，而不是从用户的计划限制中扣除，Anthropic还为开发者添加了100美元信用额度。

## Key Points | 关键要点

### System Prompt Changes | 系统提示变更
- **Blockers**: Getting weirder and weirder | 阻止器：变得越来越奇怪
- **Third-party apps**: Now draw from extra usage, not plan limits | 第三方应用：现在从额外使用量扣除，而非计划限制
- **Developer credit**: Anthropic added $100 credit for developers | 开发者信用：Anthropic为开发者添加了100美元信用

### Technical Context | 技术背景
- **CLI usage**: `claude --append-system-prompt` | CLI用法：`claude --append-system-prompt`
- **Rate limits**: Separate from user plan | 速率限制：与用户计划分开
- **Impact**: Developers affected by policy changes | 影响：开发者受政策变化影响

### Community Response | 社区响应
- **Confusion**: Users finding it difficult to understand | 困惑：用户发现难以理解
- **Workarounds**: Seeking alternatives | 变通方案：寻找替代方案
- **Transparency**: Requests for clearer communication | 透明度：要求更清晰的沟通

# Hermes Agent vs OpenClaw 全维度对比

## Summary | 摘要

Comprehensive comparison of Hermes Agent and OpenClaw. Hermes is "Agent-first", OpenClaw is "control-plane-first". Different architectures for different use cases.

Hermes Agent与OpenClaw的全面对比。Hermes是"Agent优先"，OpenClaw是"control-plane优先"。不同架构适合不同用例。

## Key Comparison | 核心对比

### Hermes Agent
- **Language**: Python
- **Architecture**: "Agent-first"
- **Focus**: Single Python Agent, procedural memory
- **Complexity**: Medium, easier to read

### OpenClaw
- **Language**: TypeScript
- **Architecture**: "Control-plane-first"
- **Focus**: Gateway, multi-agent routing, plugin ecosystem
- **Complexity**: High, more layers

### Key Differences | 关键差异
| Aspect | Hermes | OpenClaw |
|--------|--------|----------|
| Core | Single agent loop | Multi-agent orchestration |
| Multi-brain | Subagent as tools | Native multi-brain |
| Extension | ACP, MCP, cron | Plugins, nodes, apps |
